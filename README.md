# OrientDB FreeBSD Port

[OrientDB](http://orientdb.com/orientdb/) - Multi-Model NoSQL Database.

## Usage
```
fetch --no-verify-peer \
  http://github.com/wakeful/ports-orientDB/archive/master.zip \
  ; tar -xvf master.zip \
  ; rm master.zip \
  ; cd ports-orientDB-master \
  ; make clean install
```
